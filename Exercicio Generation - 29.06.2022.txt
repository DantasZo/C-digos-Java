1 O que são Metódos http: O protocolo HTTP define um conjunto de métodos de requisição responsáveis por indicar a ação a ser executada para um dado recurso sendo eles: Patch(Pequenas mudanças parciais em algum recurso), Trace( loop-back junto com o caminho para o recurso destino), Options(Descrever comunicação com o recurso de destino),Delete (remove um recurso), Put (Substitui todas as atuais representações do recurso de destino pela carga de dados da requisição), Post( utilizado para submeter uma entidade a um recurso específico, frequentemente causando uma mudança no estado do recurso ou efeitos colaterais no servidor), Head( Solicita uma resposta, sem conter o corpo da resposta), Get(solicita a representação de um recurso específico. Requisições utilizando o método GET devem retornar apenas dados).

2 Principais métodos são: Get head Post Delete Trace Put Options

3 Quais são os 2 principais métodos do HTTP? Qual é a diferença entre eles? 
Get e Post são os principais, um envia os dados e o outro retorna de volta quando solicitado.

4 O que são requisições HTTP? Quais os tipos? 
Dados são trocados entre servidor e cliente por meio de mensagens HTTP, os tipos são, Linha inicial, Cabeçalhos, linha em branco e Corpo

5.Como funciona uma requisição? O Cliente contata o servidor, e o servidor retorna uma resposta.

6.o que é o método GET? O método GET cria uma seqüência de consulta(query string) e a acrescenta à URL do script no servidor que manipula a solicitação.

7.Qual é o protocolo de internet para transferência segura, com uso de certificado digital, utilizado em sites de compras eletrônicas? é o HTTPS.

8) Quais os principais métodos de uma requisição REST? Get, Head, Put e Delete

9) Qual a diferença entre IPv4 e IPv6? O Protocolo de Internet versão 4 possui endereços no padrão 32 bits e sustenta cerca de 4 bilhões de combinações de endereços IP em todo o mundo. Já o IPv6 é a versão 6 do Protocolo de Internet, com endereços no padrão 128 bits, permitindo, portanto, cerca de 340 undecilhões de endereços.

10) Faça um resumo sobre os seguintes protocolos: http, tcp, ip, udp, smtp e ftp.

Http(Hypertext Transfer Protocol): Acesso simple utilizando uma URL, facil vizualização.

TCP: Transporte dos dados, tanto envio quanto recebimento.

IP: Endereço unico de cada maquina, usado para identificação nas redes.

FTP: Transferencia de arquivos

SMTP: envio de mensagens de email, apenas saida. 

UDP: protocolo não orientado a uma conexão, sem a verificação de integridade ou conexão.
